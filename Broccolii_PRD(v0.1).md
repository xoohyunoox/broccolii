## 1. Vision

> "Help my family prevent disease before it starts — through gentle nudging, not alarming."

브로콜리 is a family metabolic health companion that makes it easy for elderly parents, busy adults, and young children to track health data together. It replaces lectures and warnings with quiet, encouraging nudges — like a caring family member who happens to know a lot about health.

---

## 2. The Problem

- Family members do not act on verbal health advice
- Metabolic risks (high glucose, high uric acid, poor sleep, obesity) accumulate silently over years
- Existing health apps are either too clinical, too complex for elderly users, or not designed for family sharing
- Korean health data (건강검진, 영유아검진, 예방접종) is fragmented across government portals with no unified family view

---

## 3. Target Users

### Primary User: Elderly Parents (60s–70s)
- Not tech-savvy; need extreme simplicity
- Motivated by wanting to stay healthy for their family
- Likely to give up if the app feels complicated or scary
- Need large text, high contrast, minimal steps per task

### Secondary User: Adult Children (40s–50s)
- Health-conscious, tech-comfortable
- Acts as the "family admin" — sets up accounts, manages child profiles
- Wants to monitor parents and children from one place
- Will be the primary data logger (manual input after meals)

### Managed Profiles: Children (0–13)
- No independent login; managed entirely by parents
- Health data: 영유아검진 results, 학생건강검진 results, 예방접종 이력
- No wearables or devices — data entered manually or via PDF upload

---

## 4. Core Value Proposition

**Prevent disease before it starts.**

The app's reason to exist is early metabolic risk awareness — surfaced gently, not alarmingly. A user who opens the app on Day 30 should feel informed and empowered, not anxious.

---

## 5. Design Principles

1. **Nudge, don't alarm.** Replace red warnings with warm, conversational feedback. ("어제보다 혈당이 조금 높았어요. 오늘 물을 좀 더 마셔볼까요? 🥦")
2. **Elderly-first UX.** Every screen designed for a 70-year-old first. Large tap targets, minimal text, clear hierarchy.
3. **Family as the unit.** Home screen shows the whole family at a glance, not just one user.
4. **Calm aesthetics.** Soft greens, warm whites, rounded corners. No aggressive reds or flashing indicators.
5. **Progressive disclosure.** Show simple summaries first; detail is one tap away.

## 6. MVP Scope

### 6.1 Authentication
| Feature | Details |
|---|---|
| Phone number login | OTP via SMS — simple, no social account needed |
| Session persistence | Stay logged in unless manually signed out |
| Account recovery | Via phone number re-verification |

> **Deferred to v1.1:** Kakao Login, Naver Login, Apple Login

### 6.2 User Profiles
| Feature | Details |
|---|---|
| Adult profile | Name, date of birth, sex, profile photo (optional) |
| Child profiles | Parent-managed; name, date of birth, sex |
| Family view | Home screen shows all family members the adult manages |
| Profile limit | Up to 6 family members per account (MVP) |

### 6.3 Glucose Tracking (Adult)
| Feature | Details |
|---|---|
| Manual input | Log glucose value (mg/dL) with timestamp and meal context (공복/식전/식후 1시간/식후 2시간) |
| History view | List of past logs with date/time/context |
| Trend chart | Weekly line chart showing glucose over time |
| AI nudge | After each log, a gentle one-sentence feedback message generated by AI |
| Reference ranges | Display soft color bands (normal / borderline / elevated) — no hard red alerts |

> **Deferred to v1.1:** Dexcom CGM integration

### 6.4 건강검진 Data (Adult)
| Feature | Details |
|---|---|
| PDF upload | User uploads 국민건강보험 건강검진 결과 PDF |
| AI extraction | Claude reads the PDF and extracts key values (혈당, 콜레스테롤, 혈압, BMI, 요산 등) |
| Results display | Clean summary card per checkup year |
| Trend over years | Show how key values changed across checkup years |
| AI nudge | Summary feedback based on extracted values |

> **Deferred to v1.1:** CODEF API / 틸코블렛 live data sync

### 6.5 Children's Health Data
| Feature | Details |
|---|---|
| 영유아검진 | PDF upload → AI extraction of growth metrics and developmental screening results |
| 학생건강검진 | PDF upload → AI extraction of height, weight, vision, dental, scoliosis, blood results |
| 예방접종 이력 | Manual entry or PDF upload from 예방접종 도우미 |
| Display | Simple timeline view per child; growth chart for height/weight |
| Parent view | Parent sees all children's data from their home screen |

### 6.6 Notifications
| Feature | Details |
|---|---|
| Gentle nudges | Daily or post-meal reminder to log glucose (opt-in) |
| Weekly summary | "이번 주 가족 건강 요약" push notification (opt-in) |
| Tone | Always warm and encouraging, never urgent or alarming |

## 8. Technical Stack

| Layer | Technology | Reason |
|---|---|---|
| Mobile framework | React Native + Expo | One codebase for iOS + Android; large community |
| Backend & database | Supabase | Managed Postgres + auth + storage; generous free tier |
| AI feedback | Anthropic Claude API | Generates nudge messages; extracts PDF data |
| PDF parsing | Claude vision (via API) | Reads 건강검진 / 영유아검진 PDFs |
| Auth | Supabase phone auth (OTP) | Simple, no social accounts required for MVP |
| Charts | Victory Native or Recharts | Lightweight charting for React Native |
| Notifications | Expo Notifications | Cross-platform push notifications |

## 9. User Stories (MVP)

### Authentication
- As a user, I can sign up using my Korean phone number so that I don't need to remember a password.
- As a user, I can log in via OTP sent to my phone so that access is simple and secure.

### Family Profiles
- As an adult, I can create profiles for my children so that I can manage their health data without them needing their own account.
- As an adult, I can see all family members on my home screen so that I get a quick overview of everyone's health at once.

### Glucose Tracking
- As a user, I can manually log my glucose level after a meal so that I can track my metabolic health over time.
- As a user, I can see a weekly trend chart of my glucose levels so that I can notice patterns.
- As a user, I receive a gentle AI-generated message after each log so that I feel guided rather than judged.

### 건강검진
- As a user, I can upload my 건강검진 PDF so that my results are automatically read and saved without manual entry.
- As a user, I can see how my key health values have changed year over year so that I understand my long-term metabolic trajectory.

### Children's Health
- As a parent, I can upload my child's 영유아검진 or 학생건강검진 PDF so that their growth and health data is tracked in one place.
- As a parent, I can see my child's vaccination history so that I know which vaccines are upcoming.
